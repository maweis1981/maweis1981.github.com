> 灯里烧的是记忆。快烧完的时候就会变暗，然后我们换新的灯芯，把新的名字放进去。

第四章篇幅升级为**四页十六格**，并延续角色锁定：黎恩、守梦人 Moonspirit 全程同一张脸；本章还引入一位新角色——从骨灯里出来的**灰发女孩**，同样以锁定标准像贯穿之后的连载。

## PAGE 1 · 蛾时巡灯

纸月亮还没暗透，月蛾就从四面八方涌出。黎恩没回剧院，跟着守梦人走进森林深处。她沿着渐渐暗淡的银灰禁线巡灯，一句一句解释着规矩：灯里烧的是记忆，越重的记忆烧得越久、越亮。

![第四章 PAGE 1 蛾时巡灯](/assets/img/hollow-lullaby-comic/chapter4-page1.png){: width="1400" height="2234" }
_PAGE 1：月蛾涌出 → 沿禁线巡灯 → 不同温度的骨灯 → "这是规矩"。_

## PAGE 2 · 北面的灯

森林北面，骨灯更密、更矮，风过时灯罩相碰，像骨头敲击。守梦人在一盏灯前停下——里面是一个老人的名字。灯忽然晃了一下，不是风。灯罩上，裂开一道细纹。黎恩的吊坠开始发烫：「里面……有东西。」

![第四章 PAGE 2 北面的灯](/assets/img/hollow-lullaby-comic/chapter4-page2.png){: width="1400" height="2234" }
_PAGE 2：北面密林 → 停在那盏灯前 → 灯罩裂纹 → 吊坠发烫。_

## PAGE 3 · 灯里的手

裂纹扩大，一道、两道、三道。灰白的光渗出来，然后——一只孩子的手从裂缝里伸了出来，五指张开，像在等人握住。黎恩不知道自己为什么这么做，可他伸手握住了它。骨灯彻底碎裂，碎光里落下一个赤脚的女孩，缓缓睁开灰色的眼睛。

![第四章 PAGE 3 灯里的手](/assets/img/hollow-lullaby-comic/chapter4-page3.png){: width="1400" height="2234" }
_PAGE 3：孩子的手 → 他握住 → 骨灯碎裂 splash → 女孩睁眼。_

## PAGE 4 · 月亮不属于天空

女孩看着两张陌生的脸，轻声说：「月亮，不属于天空。」守梦人呼吸一滞。女孩说，灯里有一个女人一直对她说话，长得像守梦人；她在灯里待了很久，久到月亮裂开过一次、又合上。她赤脚踩在禁线上，没有留下脚印。守梦人转身走向守梦塔：「禁线不拦死人。也不拦从没活过的人。」

![第四章 PAGE 4 月亮不属于天空](/assets/img/hollow-lullaby-comic/chapter4-page4.png){: width="1400" height="2234" }
_PAGE 4：说出预言 → "长得像你" → 掌心银痕 → 走向守梦塔，月蛾默哀。_
